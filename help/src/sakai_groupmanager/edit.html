Creating and editing adhoc groups

Creating a Group

  1. Click Create a new group
  2. Provide a name for the group and, if necessary, a description
  3. Click on the Save and add members button

Editing the group name

  1. Open the group.
  2. Click on the title to reveal the edit block where you may change the text.
  3. To save, press enter or click outside the block.

Adding and editing the group description

  1. Open the group
  2. If you haven't provided a description for an existing group, click on the plus icon to reveal the edit block.
  3. If you'd like to edit the description, click on the description text to reveal the edit block where you may change the text.
  4. To save, click outside the block.
